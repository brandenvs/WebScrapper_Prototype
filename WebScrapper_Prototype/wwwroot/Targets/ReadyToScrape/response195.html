  5. Great case! - Gregor, 3 June 2022

    Really happy with the case. Very quiet. I found the wiring for the fans a bit confusing. The RGB case doesn't have a fan speed controller only an RGB controller. Powering the fans - they are 4-wire (PWM) fans. So you need 3 headers on your motherboard, or you need splitter cable(s). The fans draw up to 0.36A each, and most motherboards are rated at 1A per fan header. So to be safe, you should only pair 2 on a single motherboard header. Fan lighting - The built in RGB controller needs to be powered from SATA power. That is all you need to do to use the controller, which is already connected to the fans. If you want to power/control the fans RGB from your motherboard, you don't need the STA power, and there is a separate "ARGB" plug to connect to your motherboard.

This is fine if you've got standard CPU cooling installed, but the moment you've got a basic liquid cooling setup you're going to be using 2 fan headers right out of the box: one for the radiator fan (if you've got a one fan radiator setup) and one to run the water pump.
